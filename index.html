Audio cutter using python

8 September 2014

During the hackathon event I attended yesterday, I got to play around with some python libraries like opencv,numpy,etc. I started liking python. Today, I started learning Django and I fell in love with it immediately. I wanted to explore more of python libraries/frameworks. I came across this link which is a list of all Python libraries and MoviePy interested me. I started reading the documentation and started realising how powerful and easier it is than using a GUI or a stand-alone application. During the FSMK-Summer Camp's cultural day, just few minutes before the event, a team had brought the songs and the start and end time of it. They were full songs and I had to cut it according to the given time limits. I was clueless on which app to use. There was no internet either. Then I did it somehow using some app, I think VLC. So, I thought of using this library to write a script which does that job quickly and using only the terminal. Here's how you can use the script: 

$ python cutaudio.py filename start_time end_time y_or_n
Yes or No is for audio preview.

MoviePy is a dependency. I'm not sure if there are more. You can figure it out, I guess.
